Estimate sample size for a future experiment

Regardless of the significance of the results, we can use the estimates of variance from our mixed effects models to estiamte the sample size needed for future experiments. We will adapt the fomula from Oberg and Vitek, 2009.

We will use the median of the variances as they are highly skewed. Since we care about percent change (or log fold change if we use log-transformed data), we will use the median of the denominator for percent change.

alpha_1 <- power * FDR / (1 + (1 - FDR) * 1)
alpha_90 <- power * FDR / (1 + (1 - FDR) * 90)
alpha_50 <- power * FDR / (1 + (1 - FDR) * 50)

We will also define the FDR adjusted alpha values for different fold changes based on the number of features we expect to change. We will estimate the number of replicates for scenarios where we expect 50% and 90% of features to change. We will also estimate the number of replicates for a single uncorrected test, as would be the case in a targeted exeriment with only a single feature we test.

Let’s go over the calculation for a single uncorrected test. In order to compute the sample size we need to compute the variance of the contrast. Statisical software implementations do this for us, but we will use the it later, so its helpful to understand the math underlying it. Equations are below:

\[ \hat{L}_{\text{within subjects}} = \hat{\bar{Y}}_{Control, Medial} - \hat{\bar{Y}}_{Osteoarthritis, Medial} \\ \text{Var}(\hat{L}_{\text{within subjects}}) = 2\hat{\sigma}^2_{residual} \]

\[ \hat{L}_{\text{between subjects}} = \hat{\bar{Y}}_{Control, Medial} - \hat{\bar{Y}}_{Osteoarthritis, Medial} \\ \text{Var}(\hat{L}_{\text{between subjects}}) = 2(\hat{\sigma}^2_{residual} + \hat{\sigma}^2_{subject}) \]

The full derevation is too broad for this vignette, but let’s touch on some parameters intuitivly. The contrast estimate, \(\hat{L}\), is the difference of two estimated means, each denoted by \(\hat{\bar{Y}}\). Any individual estimated mean has both variance components from the model, \(\text{Var}(\hat{\bar{Y}}) = \hat{\sigma}^2_{residual} + \hat{\sigma}^2_{subject}\). To compute the variance of the contrast (\(\text{Var}(\hat{L})\)) that is the difference of two estimated means, we sum the variances of the estimated means and subtract the covariance of the two means multiplied by \(2\). For between-subjects contrasts, there is no covariance between means, so we subtract nothing. For within-subjects contrasts, we expect measurements from the same subject to be correlated, so the covariance between means is the variance of the subject effect, \(\hat{\sigma}^2_{subject}\), thus cancelling out the variance of the subject effect in the variance of the estimated means, leaving only the residual variance.

We can numerically verify these equations are correct using the standard error of the contrast estimate generated by emmeans. Let’s look at the first contrast test, filtered by our hypotheses of interest.

m <- lmer_means_test[[1]]

The standard error of the contrast estimate is the square root of the variance of the contrast estimate devided by the number of biological replicates, or subjects.

\[ \text{SE}(\hat{L}_{\text{between subjects}}) = \sqrt{\frac{2(\hat{\sigma}^2_{residual} + \hat{\sigma}^2_{subject})}{I}} \]

\[ \text{SE}(\hat{L}_{\text{within subjects}}) = \sqrt{\frac{2\hat{\sigma}^2_{residual}}{I}} \]

Given that \(I\) was \(4\) for the OA dataset, we can compute the standard error of the contrast estimate for both within and between subjects comparisons and compare that emmeans’s output.

I <- 4
se_between <- sqrt((2*(s2_s + s2_r))/I)
se_within <- sqrt((2*(s2_r))/I)

The \(I\) in the denominator is the number of biological replicates, or subjects. This is what we are trying to estimate, given the variance of the contrast and the desired power and FDR. As we can see from above, increaseing \(I\) decreases the standard error of the contrast estimate - decreasing the amount of signal (the estimate) needed to overcome the noise (the standard error).

Using the median variances below, we can calculate the sample size needed to detect an absolute intensity difference of 10 at 90% power, 5% alpha, and without multiple testing correction.

calc_sample_size <- function(desired_difference,
                                 alpha,
                                 power,
                                 variance_estimate_without_I){
  delta <- seq(desired_difference[1], desired_difference[2], .1)
  z_alpha <- qnorm(1-alpha/2)
  z_beta <- qnorm(power)
  aa <- (z_alpha+z_beta)
  denom = (delta/aa)^2
  #numSample <- round(variance_estimate_without_I/denom, 0)
  numSample <- variance_estimate_without_I/denom
  return(data.frame(delta,numSample))
}

Looking at these curves we can make the following useful observations:

  • Within subjects comparisons are more sensitive than between subjects comparisons, accross all correction strategies.
  • Single comparisons are the most sensitive, followed by FDR correction for 50% true negative rate, and then FDR correction for 90% true negative rate. For FDR corrected compariosns, the higher number if unchanging analytes (true negatives), the less sensitive the comparison.
  • With the OA study’s number of biological replicates, 4, we can generally detect changes larger than 40%.

Important caveats of this approach:

  • There are many assumptions in this approach, such as the residual and biolocal variance (we use the medin accross all features for each), the number of truly differentially abundent analytes (we assume 50% and 90% true negative rates, but this is unknown and can only be used to estimate the severity of the FDR correction), the demoninator of percent change (we use the median denominator, essentially analgous to the mean intensty estimate of the reference group, but this wll be different for every feature), and the power (set to 0.9).
  • FDR correction is highly sensitive to the distribution of p values - if the distribution of unadjusted p values does not adhere to range of expected distributions (uniform when all comparisons are true negatives, right skewed with some or all comparisons are true positives) the FDR correct may produce unexpected results. This may occur due to test interdependene - an inevitability in under-aggregated data occuring from the same set of biological processes. This curve assumes that the p values are distributed as expected.
  • In reality, the detectable difference of each feature is unique, becuase the intensity and variance of each feature is unique. When calculating the sample size needed for a specific analyte, we can be much more accurate by using the estiamtes specific to that feaure.
